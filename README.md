# Mega.nz-Bot
Simple Telegram Bot to Download Files From Mega.nz and Upload It to Telegram

# Features
- All Mega.nz Links supported
- No login required
- Auto Detect File Type Before Upload to Telegram
- Auth Users Only

```

```

```
python3 main.py
```
